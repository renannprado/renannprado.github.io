# Who am I?

This a blog that I believe will mostly read by people that, somehow, into computers.
That being said, my intetion here is to walk you through `who am I` in a fun, geeky way :smile:, but really I expect many people to be able to understand.

So, let's start...

`whoami` is a command in *nix systems that tells the user that is currently logged into the computer.

`finger` is a command that shows user information.

If we combine them together...

```bash
#finger $(whoami)

Login: renannprado         			Name: Renann Prado
Directory: /home/GER/hamburg        Shell: [/bin/pt-br, /bin/en, /bin/de, /bin/elvish]
On since 1990 on tty1 from /planet/earth/brazil/sao_paulo
   21 years idle
```
